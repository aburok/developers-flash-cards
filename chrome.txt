## chrome.exe
Alt+Home # Back a page.
Alt+Left Arrow # Forward a page.
Alt+Right Arrow # Display the current website in full-screen mode. Pressing F11 again will exit this mode.
F11 # Stop loading the page or a download from loading.
Esc # Zoom in or out of a page, "-" (minus) zooms out and "+" (plus) zooms in on the page.
Ctrl+(- or +) # Pressing Ctrl and any number 1 through 8 moves to the corresponding tab in your tab bar.
Ctrl+1 - Ctrl+8 # Switch to last tab.
Ctrl+9 # Reset browser zoom to default.
Ctrl+0 # This combination is used to quickly complete an address. For example, type "computerhope" in the address bar and press Ctrl+Enter to get https://www.computerhope.com.
Ctrl+Enter # Open the Clear browsing data window to quickly clear private data.
Ctrl+Shift+Delete # Toggle the bookmarks bar between hidden and shown.
Ctrl+Shift+B # Select everything on a page.
Ctrl+A # Add a bookmark for the page currently opened.
Ctrl+D # Open the "find" bar to search text on the current page.
Ctrl+F # Open a file in the browser.
Ctrl+O # Open the Bookmark manager.
Ctrl+Shift+O # Open browser history in a new tab.
Ctrl+H # Display the downloads window.
Ctrl+J # Moves your text cursor to the omnibox so that you can begin typing your search query and perform a Google search.
Ctrl+K or Ctrl+E # Move the cursor to the browser address bar and highlight everything in it.
Ctrl+L, Alt+D # Open New browser window.
Ctrl+N # Open a new window in incognito (private) mode.
Ctrl+Shift+N # Print current page or frame.
Ctrl+P # Refresh the current page or frame.
Ctrl+R or F5 # Opens the Save As window to save the current page.
Ctrl+S # Opens a new tab.
Ctrl+T # View a web page's source code.
Ctrl+U # Closes the current tab.
Ctrl+W # Closes the currently selected window.
Ctrl+Shift+W # This combination reopens the last tab you've closed. If you've closed multiple tabs, you can press this shortcut key multiple times to restore each of the closed tabs.
Ctrl+Shift+T # Moves through each of the open tabs going to the right.
Ctrl+Tab # Moves through each of the open tabs going to the left.
Ctrl+Shift+Tab # Open a link in a new tab in the background.
Ctrl+Left-click # Open a link in a new tab and switch to the new tab.
Ctrl+Shift+Left-click # Open the browser tab to the right.
Ctrl+Page Down # Open the browser tab to the left.
Ctrl+Page Up # Moves down a page at a time.
Spacebar # Moves up a page at a time.
Shift+Spacebar # Go to top of page.
Home # Go to bottom of page.
End # Display all previous text entered in a text box and available options on a drop-down menu.
